# FLEKS Data Format

## Field

* IDL format with files ended with "out"
* AMReX built-in format, whose directory name is assumed to end with "_amrex"

## Particle

### PIC Particle

Stored with AMReX built-in format. Currently the particle data also comes with the EM field data under the same directory.

### Test Particle

Stored with FLEKS native binary data format.
