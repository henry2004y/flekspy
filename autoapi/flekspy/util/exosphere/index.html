Parameters:

  • max_altitude (float, optional) – Maximum altitude for the plot [m]. The default is 10000e3.

  • num_points (int, optional) – Number of points to use in the plot. The default is 1000.

  • ax (matplotlib.axes.Axes, optional) – An existing Axes object to plot on. If None, a new figure and axes will be created. The default is None.

  • **kwargs – Additional keyword arguments to be passed to ax.plot.
