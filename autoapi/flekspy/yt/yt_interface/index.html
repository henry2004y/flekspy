This is a generic field container. It contains a list of potential derived fields, all of which know how to act on a data object and return a value. This object handles converting units as well as validating the availability of a given field.

Plot 2D slice

Parameters:

  • norm (string, one of 'x', 'y' or 'z'. Norm direction of the slice.)

  • cut_loc (float. The location of the slice.)

  • vars (a list or string of plotting variables.) – Example: “Bx rhos0” or [“Bx”, “rhos0”]

  • unit_type (The unit system of the plots. "planet" or "si".)

Examples

>>> splt = ds.plot_slice("y", 0.0, vars)
>>> splt.display()
